歡迎使用本網站獲取爬蟲資料,請依照以下規則獲取您想要的資料內容 API address:pccuapi.servehttp.com

GET /all---get所有的資料
GET /loc/{地點}---get特定地點(建國,忠孝,延平,台中,高雄)

Example

pccuapi.servehttp.com/loc/建國

POST /all---輸入要新增的欄位值(須在POSTMAN裡操作)

Example

pccuapi.servehttp.com/all

GET /hour/{判斷式}---get(>,=,<)時間的資料

Example

pccuapi.servehttp.com/hour/<=100

GET /hours/{時間1}.{時間2} ---get(between時間1 and 時間二)時間的資料

Example

pccuapi.servehttp.com/hours/100.200

GET /price/{判斷式}---get(>,=,<)價錢的資料

Example

pccuapi.servehttp.com/price/<=10,000

GET /prices/{價錢1}.{價錢2} ---get(between價錢1 and 價錢二)價錢的資料

Example

pccuapi.servehttp.com/prices/10,000.20,000

DELETE /{id}---找到id並刪除(須在POSTMAN裡操作)

Example

pccuapi.servehttp.com/3382

PATCH /{id}---找到id並更改(須在POSTMAN裡操作)

Example

pccuapi.servehttp.com/3381
